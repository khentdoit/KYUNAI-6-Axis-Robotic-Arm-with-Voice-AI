\## ✨ Project Overview



The \*\*KYUN Robotic Arm Web Controller\*\* is a complete, production-ready web application built with Flask and WebSockets. It provides an intuitive and futuristic interface to control a physical 6-axis robotic arm. The system supports \*\*multi-user collaboration\*\*, \*\*voice AI\*\*, \*\*sequence recording\*\*, and a stunning \*\*3D real-time visualization\*\*.



This project started as a simple serial controller and has evolved into a feature-packed platform, demonstrating my ability to build integrated hardware-software solutions.

\## 🚀 Key Features



\### 🎮 Control \& Interface

\- \*\*6-Axis Control\*\*: Potentiometer knobs for individual motor control.

\- \*\*Real-time 3D Visualization\*\*: Interactive, drag-to-move 3D model (Futuristic Pink Design).

\- \*\*PID Motor Control\*\*: Smooth, precise servo movements.

\- \*\*Emergency Abort \& Global Reset\*\*: Instant stop and home position return.



\### 👥 Multi-User System

\- \*\*Editor/Viewer Mode\*\*: Only one editor controls the physical arm at a time.

\- \*\*Real-time Sync\*\*: All connected devices see the same 3D model state.

\- \*\*Role Request System\*\*: Users can request to become the editor.



\### 🎤 Voice AI System

\- \*\*Voice Recognition\*\*: Control the arm using speech commands.

\- \*\*Text-to-Speech Responses\*\*: AI responds with voice feedback.

\- \*\*Two Tones\*\*: Professional (calm) or Sassy (energetic) personality.



\### 💾 Memory \& Sequences

\- \*\*12 Memory Slots\*\*: Save and load sequences of poses.

\- \*\*Teach Pendant Mode\*\*: Record movements in real-time.

\- \*\*Sequence Execution\*\*: Run saved sequences with loop options and smooth easing.



\### ⚙️ Hardware Integration

\- \*\*Serial Communication\*\*: Connects to Arduino via COM port.

\- \*\*Collision Zones\*\*: Set min/max angle limits for each joint to prevent damage.

\- \*\*Simulation Mode\*\*: Test without hardware connected.



\### 📊 Analytics Dashboard \& Chat Terminal

\- \*\*Move \& Sequence Counter\*\*: Tracks usage and performance.

\- \*\*AI Conversation\*\*: Ask questions about the developer/team.

\- \*\*Resizable Chat Panel\*\*: Command interface with history.



\### 🔒 Safety \& Security

\- \*\*Collision Zone Limits\*\*: Prevents mechanical damage.

\- \*\*Editor-Only Control\*\*: Prevents conflicting commands.

\- \*\*Emergency Stop\*\*: Immediate halt to all operations.
